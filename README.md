# Sajjalashree Surgical & Medical Equipments

A premium, modern web application for **Sajjalashree Surgical**, a leading supplier of surgical instruments, hospital furniture, and specialized medical equipment based in Bellary, Karnataka.

![Sajjalashree Surgical Banner](https://images.unsplash.com/photo-1519494026892-80bbd2d6fd0d?q=80&w=1200)

## 🚀 Key Features

### 📦 Medical Product Catalog
- **Diverse Range**: Surgical instruments, diagnostic devices, laboratory equipment, and hospital furniture.
- **Advanced Filtering**: Categorized browsing with search functionality and availability toggles.
- **Premium Visualization**: High-resolution product images and detailed feature breakdowns.

### 🔄 Equipment Rental System (NEW)
- **Specialized Hire**: Rent essential medical devices like Blood Glucose Monitors (BGM), Muscle Stimulators, and Lymphedema Compression Pumps.
- **Flexible Terms**: Easy access to weekly and monthly hire rates for clinics and home care.
- **Badge Integration**: Clear "Available for Hire" status on all rentable products.

### 🛠️ Specialized Technical Services
- **High-End Repair**: Expert maintenance for complex medical machinery, including **Heart-Lung Machines**.
- **Quality Assurance**: Certified testing and calibration to meet ISO and regulatory standards.
- **Emergency Support**: Direct inquiry pathways for critical bio-medical technical support.

### 🎨 Modern UI/UX
- **Premium Aesthetics**: Apple-inspired design with glassmorphism, smooth gradients, and immersive micro-animations.
- **Responsive & Dynamic**: Fully optimized for mobile, tablet, and desktop viewing.
- **Theme Support**: Seamless transitions between Dark and Light modes.

## 🛠️ Technology Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Routing**: [React Router](https://reactrouter.com/)

## 🏁 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## 📂 Project Structure

- `src/components`: Reusable UI components (GlassCard, ProductCard, etc.)
- `src/pages`: Main page components (Home, Products, Services, About, Contact)
- `src/data`: JSON data stores for products and services
- `src/assets`: Global styles and static assets
- `public/images`: Product and site imagery
